1. Quel est le problème ?

Les Universités françaises opèrent de nombreux sites Web, avec un panel hétérogène de technologies. Ces sites présentent des performances énergétiques très variables, et représentent un trafic Web non négligeable.

2. Quelle méthodologie de mesure ?

Nous avons mesuré avec notre outil de diagnostic écologique l'impact carbone de deux pages pour chaque site d'Université, la page d'accueil et la page de mentions légales. Nous avons choisi la moyenne de ces deux valeurs, que nous avons multiplié par le nombre de pages vues, sur la base des données fournies par Similar Web.

3. Les chiffres sont-ils fiables ?

Les chiffres présentent des incertitudes, qui sont détaillées sur chaque page d'Université. Toutefois, la réalité est probablement au-dessus de ces chiffres, car nous mesurons seulement le site principal de chaque Université. Dans la réalité, une Université opère de très nombreux sites, directement ou indirectement : composantes, laboratoires, bibliothèques, formations, projets spécifiques...

4. Comment agir ?

Chez noesya, nous développons Osuny, une plateforme numérique en source ouverte pour les Universités & les écoles. Nous avons mesuré l'impact carbone du site réalisé avec l'Institut Universitaire de Technologie Bordeaux Montaigne , et établi une comparaison systématique pour évaluer le CO2 évitable : plus de 9000 tonnes eq CO2.

5. Donc, concrètement ?

Vous faites des sites Web (développement, recherche utilisateurs, design interactif, stratégie, gestion de projet, ergonomie, accessibilité, éco-conception...) et vous avez envie d'avancer sur ce sujet ?
